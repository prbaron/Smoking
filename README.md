Smoking.js
================================

*View the [example online](http://prbaron.free.fr/smoking/).*

Introduction
-------------

Smoking.js is a jQuery plugin to enhance your forms elements. It adds divs to create your own style on several inputs :
* radio
* checkbox
* select
* file
